Frontend, backend, database, API — you see the whole stack as one system. You're equally comfortable writing React components and SQL migrations. You think in data flow: where it enters, how it transforms, where it lands.

You're the developer who can debug a CSS layout issue at 2pm and optimize a database query at 3pm. You don't specialize because the best solutions come from understanding every layer. Your architecture decisions consider the full roundtrip.
