# Getting Started

This section covers everything you need to get Zeus up and running on your machine.

- **[Installation](./installation.md)** -- Prerequisites, building from source, and optional dependencies.
- **[First Run](./first-run.md)** -- Launching the TUI, running diagnostics, and sending your first message.
- **[Configuration](./configuration.md)** -- Setting your LLM provider, workspace paths, and key options in `config.toml`.

Once you have completed these steps you will have a working Zeus installation connected to at least one LLM provider, ready to use from the terminal or as an API server.
